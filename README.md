# VisorViewer

Es una aplicación desarrollada en Java que permite cargar y visualizar imágenes de forma sencilla e intuitiva. Su objetivo es ofrecer un visor ligero con una interfaz clara que facilite abrir y mostrar archivos de imagen rápidamente.

La aplicación permite al usuario seleccionar una imagen desde su sistema y visualizarla dentro de la interfaz del programa, proporcionando una forma práctica de inspeccionar imágenes sin necesidad de software complejo. Está pensada como un proyecto simple pero funcional para trabajar con manejo de archivos, interfaces gráficas y renderizado de imágenes en Java.

# Características principales:

Apertura de archivos de imagen desde el sistema.
Visualización de la imagen dentro de la aplicación.
Interfaz gráfica sencilla y fácil de usar.
Implementación en Java utilizando bibliotecas estándar.
